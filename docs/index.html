InstructSAM

A Training-Free Framework for Instruction-Oriented Remote Sensing Object Recognition

GitHub arXiv

About

InstructSAM is a training-free framework for Instruction-Oriented Object Counting, Detection, and Segmentation (InstructCDS). It addresses remote sensing tasks with open-vocabulary, open-ended, and open-subclass settings.

Features

  • Training-free framework for remote sensing object recognition.
  • Supports open-vocabulary, open-ended, and open-subclass settings.
  • Integration with SAM2 and CLIP models.
